# Volume Renderer
This repository is volume data renderer for image to mesh reconstruction work.

# Requirement
Install pytorch for individual environment.

### Test Environment
cuda=12.4, python=3.10, pytorch = 2.6.0
```
pip install -r requirements.txt
```

Pytorch Install 
https://pytorch.org/get-started/previous-versions/
(nvcc -V 터미널에 입력하면 버전 볼 수 있음. 쿠다와 호환되는 버전 복붙해서 터미널에 입력)
